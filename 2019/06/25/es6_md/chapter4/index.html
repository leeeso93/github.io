4.2.1 제어문의 예외

제어문의 실행 방식을 바꿈
break :루프 중간에 빠져나감
continue : 루프 다음 단계로 바로 건너뜀
return : 루프문 무시 함수 빠져나감
throw : 반드시 처리해야할 예외를 발생(제어문 밖에서도 작용)
