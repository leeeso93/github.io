Angular chapter6 Angular의 파일 구조와 처리 흐름

angular CLI로 프로젝트를 생성하면 아래와 같인 파일 구조의 스캐폴딩이 생성된다.

my-app/
├── .git/
├── e2e/
├── node_modules/
├── src/
├── .editorconfig
├── .gitignore
├── angular.json
├── package-lock.json
├── package.json
├── README.md
├── tsconfig.json
└── tslint.json

  1. src 폴더

    src 폴더는 angular의 모든 구성요소, css,이미지, 폰트, 설정 파일 등 애플리케이션 필수 파일을 담는다.

  2. 기타 설정 파일

    src 폴더 밖의 파일은 테스트, 빌드, 배포 등을 위한 각종 설정 파일이다.
